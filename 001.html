Python 中的列表和 C++中的数组和 vector 有什么差异？

概念差异

功能差异：

  • Python 中的列表（List）是 动态数组，可以容纳不同类型的元素，并且可以动态调整大小。列表支持各种操作，如添加、删除和索引等。
  • C++ 中的数组（Array）是一种 静态数据结构，用于存储相同类型的元素。数组的大小在声明时确定，无法改变。数组可以进行索引操作，但不支持动态添加或删除元素。
  • C++ 的 vector 是一个 动态数组类模板，类似于 Python 的列表。它可以容纳不同类型的元素，并且可以动态调整大小。vector 支持各种操作，包括添加、删除、索引和排序等。

实现差异：

  • Python 中的列表（List）是通过 动态链表 实现的，是对源码中的对象（PyListObject）的抽象。每个列表元素都是一个对象，包含值和指向下一个元素的指针。
  • C++ 中的数组（Array）是一块 连续的内存区域，元素的访问是通过索引和偏移量进行的，内存分配是在编译时或运行时完成的。
  • C++ 的 vector 也是通过 动态分配内存 来实现的，但它封装了数组的操作，并提供了更多的功能和安全性，如自动调整大小、边界检查和内存管理等。

特点比较

空间效率方面：

  • Python 中列表（List）由于不同的列表可能包含不同数量的元素，因此 Python 的列表在创建时被设置为 变长对象。列表对象内部包含了插入、删除等方法，可以在运行时动态调整维护的内存和元素，具有 较高的空间效率。
  • C++ 中创建数组（Array）时需要预先指定数组的容量大小，并根据大小来分配内存。即使只需要存储一个元素，也需要为所有的元素预先分配内存，因此 C++ 中的数组 空间效率不高。
  • C++ 的 vector 类在内存分配方面相对灵活，可以 动态调整容量。它通常会分配比实际元素数量更多的内存作为缓冲区，以便在需要时可以扩展容量而不必频繁重新分配内存。这种预分配的策略可以提高空间效率，减少内存重新分配的次数，因而同样拥有 较高的空间效率。

长度方面：

  • Python 中列表（List）是 可变的，可以通过添加或删除元素来改变列表的长度。
  • C++ 中 除了动态数组（Dynamic Array） 之外，其他类型的数组（Array）都是 不可变的，一旦初始化大小就无法改变。
  • C++ 的 vector 类是 可变长度的，可以在运行时动态调整大小，可以通过添加或删除元素来改变 vector 的长度。

数据类型方面：

  • Python 中列表（List）的元素 可以是不同的数据类型，它具有灵活性。
  • C++ 中数组（Array）的元素 必须是相同的数据类型，并且在数组初始化时就已经确定了。
  • C++ 的 vector 类 可以存储不同类型的元素，类似于 Python 的列表。它使用模板来实现，可以在声明 vector 时指定元素的类型，并在运行时存储不同类型的对象。

综上所述，C++ 的 vector 在空间效率方面相比数组（Array）更高，可以动态调整容量以减少内存重新分配的次数。它也可以存储不同类型的元素，并且长度是可变的，具有更大的灵活性。然而，与 Python 的列表（List）相比，C++ 的 vector 在灵活性和动态调整大小的功能上略有限制，因为它需要在编译时确定元素类型和容器的大小。

代码范例：遍历元素

在 Python 中使用列表（List）遍历元素：

my_list = [1, 2, 3, 4, 5]

# 使用 for 循环遍历列表元素
for element in my_list:
    print(element)

在 C++ 中使用数组（Array）遍历元素：

#include <iostream>

int main() {
    int my_array[] = {1, 2, 3, 4, 5};
    int size = sizeof(my_array) / sizeof(my_array[0]);

    // 使用 for 循环遍历数组元素
    for (int i = 0; i < size; i++) {
        std::cout << my_array[i] << std::endl;
    }

    return 0;
}

在 C++ 中使用 vector 遍历元素：

#include <iostream>
#include <vector>

int main() {
    std::vector<int> my_vector = {1, 2, 3, 4, 5};

    // 使用范围-based for 循环遍历 vector 元素
    for (int element : my_vector) {
        std::cout << element << std::endl;
    }

    return 0;
}

适用 Python 中的 lambda 函数有何优势，请举例说明。

在 Python 中，lambda 函数是一种匿名函数，它可以在需要函数对象的地方使用，并且通常用于简化代码和处理简单的函数逻辑。

lambda 函数的优势

  • 简洁性：lambda 函数可以在一行代码内定义函数，避免了传统函数定义的繁琐语法。它们通常用于处理简单的函数逻辑，使代码更加紧凑和易读。
  • 避免命名冲突：由于 lambda 函数是匿名的，不需要命名，可以避免函数命名冲突的问题。特别是在需要传递简单函数作为参数的情况下，使用 lambda 函数更加方便。
  • 函数式编程：lambda 函数可以与函数式编程结合使用，例如在 map、filter 和 reduce 等函数中。它们可以作为参数传递给这些函数，提供灵活的功能扩展。

使用范例

加法计算：

add = lambda x, y: x + y
result = add(3, 5)
print(result)  # 输出: 8

排序：

students = [
    {'name': 'Alice', 'age': 20},
    {'name': 'Bob', 'age': 18},
    {'name': 'Charlie', 'age': 22}
]
students.sort(key=lambda student: student['age'])
print(students)
# 输出: [{'name': 'Bob', 'age': 18}, {'name': 'Alice', 'age': 20}, {'name': 'Charlie', 'age': 22}]

条件筛选：

numbers = [1, 2, 3, 4, 5, 6]
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # 输出: [2, 4, 6]
